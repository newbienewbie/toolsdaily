# 领料调拨记账辅助脚本

## 背景
由马勇替我们单位定制的仓储管理系统经常会莫名其妙丢数据、重复插入记录，但是由于经费原因，马勇不再维护。每次做完系统调拨后都必须手工登记到excel，非常耗时耗力，遇到多人同时操作excel时，还需排队。
于是开发这个小脚本，实现登记台账自动化。考虑到办公室其他人不会使用其他数据库，于是采用Access数据库。

## 原理
马勇开发的客户端会动态合成、加载调拨单，我修改其printDBD.txt文件，利用javascript向其他服务器post请求，我假设的服务端接收到请求后写入Access数据库。

## 用法
客户端：把printDBD.txt覆盖马勇的调拨系统中的HTML/printDBD.txt
服务端：
* 假设网站服务器，地址为10.16.165.167
* Access数据库在I盘的根目录下
* 接收客户端请求的服务端文件为：wzdb.php（地址是http://10.16.165.167/wzdb.php）
